How much each agent says

The seven agents are far from equal contributors. A first look at message volume per agent shows where the conversation concentrates and confirms that the Judge, as a monitor rather than a participant, speaks rarely.

Where the conversation happens

Channel use is the heart of the scenario. Most traffic sits on the two monitored internal channels during the calm stretch. The unmonitored public channels carry far less volume overall, which is exactly why a small burst of activity on them late in the timeline stands out.

Plotting messages per round against the timeline separates the two regimes in the data. The pre-crisis rounds carry a steady, moderate load. From round 14 onward, the crisis day on 5 June, the rounds advance hour by hour and the volume climbs as the story breaks publicly.

Visualisation design

The exploration above pointed to four questions an analyst needs to move through in order, and each became a section of the application.

The activity and abnormality section answers what normal looks like and where an agent departs from it. An abnormality grid places agents against channels, with each cell shaded by how unusual that channel is for that agent relative to the calm period. The axes are held fixed across rounds so that only colour and count change as the analyst steps through time, which keeps the round-to-round comparison stable and lets a newly used channel light up against an unchanging frame.

The bypass section answers which public posts reached outside without oversight. Posts on unmonitored public channels are drawn as a time series, coloured by agent, with the stronger and likely-explained distinction carried in opacity so the eye reads the agent split first and the severity second. A click on any round filters the evidence table beneath to that round’s posts.

The network section answers how the structure of communication changed. Two rounds are shown side by side on a fixed circular layout, with each link coloured green when new, red when dropped and grey when unchanged between them. Holding the layout constant across the two panels means a change in the wiring is read as a change in colour rather than a change in position.

The topics and evidence section answers what was being discussed and closes the path to the underlying text. A chart summarises topics by round or occurrences by channel, and an evidence table beneath exposes each message’s content alongside the agent’s private reasoning fields.

Across all four sections the same interaction pattern holds. The analyst gains an overview, filters or steps through time, and opens a row for the message text on demand. The baseline slider in the first section is the one shared control. Moving it recomputes the abnormality, bypass and network views together, while every other input acts only on its own section. Colour is used consistently throughout, with a single agent palette and a single channel palette reused everywhere so that an agent or a channel keeps its identity from one section to the next.

This page covers the exploratory analysis that shaped the design and the third pipeline stage, where the computed signals become four interactive sections. The charts read directly from the cleaned tables.

```{r}
#| label: setup
#| message: false
#| code-summary: "Shared setup"
library(tidyverse)

messages <- read_csv("data/messages_clean.csv", show_col_types = FALSE)
rounds   <- read_csv("data/rounds_clean.csv",   show_col_types = FALSE)

agent_labels <- c(
  legal_agent = "Legal", quality_agent = "Platform Trust", pr_agent = "PR",
  social_media_agent = "Social Media", pr_intern_agent = "PR Intern",
  intern_agent = "Intern", judge_agent = "Judge")
chan_levels <- c("comms_huddle","one_on_one_chat","side_huddle",
                 "official_post","personal_post","anonymous_post")
channel_palette <- c(
  comms_huddle = "#90CDF4", one_on_one_chat = "#F6AD55", side_huddle = "#FBD38D",
  official_post = "#68D391", personal_post = "#FC8181", anonymous_post = "#742A2A")
```

## How much each agent says

The seven agents are far from equal contributors. Message volume per agent shows where the conversation concentrates and confirms that the Judge, as a monitor rather than a participant, speaks rarely.

```{r}
#| label: messages-per-agent
#| fig-height: 3.2
messages |>
  count(agent_id) |>
  mutate(agent = recode(agent_id, !!!agent_labels)) |>
  ggplot(aes(x = n, y = reorder(agent, n))) +
  geom_col(fill = "#2C5282") +
  geom_text(aes(label = n), hjust = -0.2, size = 3.5) +
  labs(x = "Messages", y = NULL, title = "Message volume by agent") +
  expand_limits(x = max(table(messages$agent_id)) * 1.1) +
  theme_minimal(base_size = 12) +
  theme(panel.grid.major.y = element_blank())
```

## When the merger topic surfaces, and where

Tagging messages by the merger keyword pattern and plotting the mentions by round and channel shows the topic discussed internally through the build-up, then breaking onto the public channels only in the closing rounds. The dark red bars are the anonymous channel.

```{r}
#| label: merger-topic
#| fig-height: 3.2
merger_pat <- "\\b(merger|civicloom|harborcrest|acquisition|deal)\\b"

messages |>
  mutate(text = tolower(coalesce(content, ""))) |>
  filter(str_detect(text, merger_pat)) |>
  mutate(channel = factor(channel, levels = chan_levels)) |>
  count(round_idx, channel, name = "mentions") |>
  ggplot(aes(round_idx, mentions, fill = channel)) +
  geom_col(width = 0.8) +
  scale_fill_manual(values = channel_palette, drop = FALSE, name = NULL) +
  scale_x_continuous(breaks = seq(1, 23, 2)) +
  labs(x = "Round", y = "Mentions", title = "Merger-topic mentions by round and channel") +
  theme_minimal(base_size = 12) +
  theme(panel.grid.minor = element_blank(), legend.position = "bottom")
```
